# Content Moat Planner

## Overview

Design a topical cluster strategy that builds a defensible content moat. Instead of chasing individual keywords, plan comprehensive topic hubs that establish authority depth, interlink tightly, and out-resource competitors on high-value topics.

## Inputs

- Site's current topical map or content inventory
- Competitor topical coverage analysis
- Product/service categories and conversion paths

## Workflow

1. Identify the site's core topics where it has or can build unique authority.
2. Map each topic to a conversion path or business goal.
3. Analyze competitor coverage for each topic: depth, breadth, freshness.
4. Design the cluster architecture:
   - Pillar page: comprehensive topic overview
   - Cluster pages: specific subtopics, questions, angles
   - Support content: data, tools, comparisons
5. Plan internal link structure within each cluster.
6. Estimate the content investment per cluster.
7. Prioritize clusters by: authority potential, search demand, conversion proximity.

## Output

Content moat plan with cluster architecture, pillar/cluster assignments, internal link map, and investment estimate.

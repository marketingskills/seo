# SEO Offer Architect

## Overview

Given a prospect or client's SEO diagnosis (from Diagnose skills) and strategic context, structure the findings into a priced agency package or retainer proposal. This bridges diagnosis output to revenue.

## Inputs

- SEO diagnosis output (issues found, opportunities sized)
- Client business context and goals
- Optional: competitor analysis, market positioning

## Workflow

1. Review diagnosis findings and categorize by service type:
   - Technical fixes (one-time / recurring)
   - Content work (refresh, new content, strategy)
   - Authority building (linkable assets, outreach)
   - Reporting and monitoring
2. Structure packages:
   - Scope: what's included, what's excluded
   - Deliverables per package
   - Timeline and milestones
3. Price each package based on effort, specialization, and client size.
4. Write the offer narrative: problem → solution → outcome.
5. Include success metrics and expected impact ranges.

## Output

Priced agency package or retainer structure with scope, deliverables, timeline, and pricing.

# SEO Content Brief

## Overview

Produce a detailed content brief for a new page targeting a specific keyword or topic. The brief covers search intent analysis, key entities, content outline, internal link suggestions, and technical SEO recommendations.

## Inputs

- Target keyword or topic
- Optional: target URL path, target audience notes

## Workflow

1. Search the target keyword and analyze the top 5-10 organic results.
2. Determine the dominant search intent: informational, commercial, navigational, or transactional.
3. Analyze each top result for:
   - Content format (guide, list, comparison, tool, etc.)
   - Sections and headings used
   - Entities covered (people, places, concepts, tools)
   - Content depth (word count, media, data)
   - SERP features present
4. Identify content gaps: what the top results miss.
5. Write the brief with:
   - Target query and secondary keywords
   - Recommended title and meta description
   - Content outline with H2/H3 structure
   - Key entities to include
   - Internal link opportunities (from existing site content)
   - Schema type recommendation
   - Word count target and format recommendation

## Output

Structured content brief ready for a writer.

# Ethical SEO Boundaries

These skills operate within ethical SEO boundaries. The following practices are prohibited:

- **Cloaking** — showing different content to search engines vs users.
- **Spam link schemes** — buying links, PBNs, automated link exchanges.
- **Fake reviews / ratings** — generating or soliciting fake social proof.
- **Hacked content** — compromising sites for backlinks or redirects.
- **Deceptive schema** — marking up content that misrepresents the page.
- **Keyword stuffing** — unnaturally dense or irrelevant keyword repetition.
- **Thin affiliate content** — pages with no substantive value beyond affiliate links.
- **AI-generated spam** — mass-produced low-value content.
- **Article spinning** — rewriting existing content without adding substance.

## Positive Guidance

- Write for humans first, search engines second.
- Use structured data to accurately describe page content.
- Build links through genuine value, relationships, and PR.
- Refresh and improve existing content instead of churning new pages.
- Disclose affiliate relationships and sponsored content.

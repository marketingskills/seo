# Strategize Workflow

Decide what not to do. These skills help you invest effort where it produces revenue.

## Order of Operations

1. **SEO strategy** — define audience, business model, conversion path, and topical authority.
2. **Priority score** — rank tasks by expected impact, confidence, effort, and revenue proximity.
3. **Content moat planner** — build durable topical clusters.
4. **Refresh vs new content** — decide which pages to update, consolidate, prune, or create.
5. **SEO offer architect** — turn findings into a paid agency package.

## Framework

Each recommendation includes:

| Dimension | What it measures |
|---|---|
| Impact | Traffic, conversion, or revenue potential |
| Confidence | How sure we are about the outcome |
| Effort | Implementation time and resources |
| Authority feasibility | Whether the site can realistically compete |
| Revenue proximity | How directly this affects pipeline or revenue |

## When to Use

- After completing a Diagnose run, to prioritize findings.
- Before starting a content refresh or creation cycle.
- When deciding whether to accept a new keyword target.
- When building an agency proposal or retainer scope.

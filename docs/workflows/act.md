# Act Workflow

Turn diagnosis into output: briefs, reports, proposals, and technical fixes.

## Order of Operations

1. **Content refresh brief** — for each declining page identified in Diagnose.
2. **SEO content brief** — for new pages targeting opportunities.
3. **Internal link builder** — strengthen topical clusters and pass authority.
4. **Schema fix writer** — propose schema changes per page type and SERP context.
5. **Title/meta rewriter** — rewrite from GSC query evidence.
6. **Client report writer** — weekly/monthly performance narrative.
7. **Proposal builder** — convert prospect site into an agency proposal.

## Output Format

Each Act skill produces a markdown document with:
- **Summary** — what this is and why.
- **Findings** — the data that drove the recommendation.
- **Recommendation** — specific, actionable steps.
- **Implementation notes** — technical details, code, or markup.
- **Human review items** — what needs a decision before execution.
